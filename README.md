# Sizzy Landing Page

Modern, aesthetic landing page for Sizzy - The Browser for Web Developers.

## Features

- Built with React 19 + Vite + Tailwind CSS v4
- Framer Motion animations
- Interactive device preview demos
- Dark mode design
- Fully responsive

## Getting Started

```bash
# Install dependencies
pnpm install

# Start development server
pnpm dev

# Build for production
pnpm build

# Preview production build
pnpm preview
```

## Tech Stack

- **Framework:** React 19 + Vite
- **Styling:** Tailwind CSS v4
- **Animations:** Framer Motion
- **UI Components:** Radix UI
- **Icons:** Lucide React
